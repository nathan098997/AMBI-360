Como testar:

  1. Clique em um dos links acima
  2. O projeto deve abrir diretamente no viewer
  3. Se houver hotspots, você pode navegar entre eles
  4. Use o botão "Compartilhar" para gerar novos links

Problemas comuns:

  • Link não funciona: Verifique se o arquivo index.html está na pasta frontend/
  • Imagens não carregam: Verifique a conexão com a internet (usa imagens do Pannellum)
  • Erro de decodificação: O link pode estar corrompido
